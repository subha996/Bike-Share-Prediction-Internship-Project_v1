About and Contributor


The Project is about build a system where the user can predict expected customer for a particular day, based on the weather condition at the place of the client store and date information. From the system the client can fill in some weather information like temperature, wind speed, humidity, from and some date information like month, season, e.t.c, based on that a output will be shown. The system is design such a way that if user choose not to fill any information then all the required information will be capture automatecly on the user current location. And will give output completely automaticly. From this information user can decied there business plan, that how much customer he can expected for a particular day, so they can arrange the other needs that are provide for there business.
